# My Usage

Next.js application with GitHub OAuth (Auth.js), Neon Postgres via `@auth/neon-adapter`, and shadcn/ui.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Copy `.env.example` to `.env.local` and configure environment variables locally.

Architecture and schema details are summarized in [`auth-plan.md`](auth-plan.md).
